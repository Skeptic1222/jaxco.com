The UMG Portal Intranet program was one of the largest projects for the Strategic Web Application Technology team. Each of the varied Universal business units wanted their own custom intranet, but as we built them, we realized that essentially the same functionality was being re-hashed with a new design or front-end differentiating one site from the next.

The team set to work on a solution that would deliver the features that each of the clients wanted, while still affording them individuality in look and feel.

One of my major responsibilities was to construct new designs for the Portal program based upon the framework and guidelines set forth in the common codebase project. Much of the interior work was done with custom stylesheets. The program includes well over 100 class entries, so the smallest details could be adjusted. As the design lead for the project, I worked closely with the lead developers and various project managers to ensure that the clients' creativity and individuality would not get lost in a program that was designed to cut costs and development time.

Through the use of custom html, JavaScript and asp code, various front-end effects were achieved. As all of these sites are intranets, I am unable to link directly them, but please feel free to view the design samples provided.
